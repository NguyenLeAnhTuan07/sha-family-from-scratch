# SHA Family – From Scratch Implementation

This project contains manual implementations of several cryptographic hash functions, including:

- SHA-1
- SHA-256
- SHA-512
- SHA-3

All algorithms are implemented from scratch without using built-in hashing libraries.

## Purpose

The goal of this project is to deeply understand the internal structure of cryptographic hash functions, including:

- Message preprocessing
- Padding mechanisms
- Word expansion
- Compression functions
- Bitwise operations and modular arithmetic

## Educational Focus

This repository is intended for:

- Learning cryptographic hash internals
- Understanding Merkle–Damgård construction
- Exploring sponge construction (SHA-3)
- Practicing low-level bit manipulation

## Warning

This implementation is for educational purposes only.
Do NOT use it in production systems.
